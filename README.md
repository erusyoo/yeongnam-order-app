# 🚚 영남물류 스마트 발주 시스템 (Demo)

모바일 환경에서 현장 파견 직원(영업사원)이 팩스/전화 없이 간편하게 발주할 수 있도록 제작된 Streamlit 기반 B2B 모바일 발주 시스템입니다.

## 📌 주요 기능 (Features)
- **📱 모바일 터치 최적화 UI:** 현장 작업장갑 착용 상태에서도 조작 가능한 대형 버튼 레이아웃
- **➕➖ 직관적인 수량 입력:** 스크롤 튐 현상을 방지한 모바일 최적화 스텝핑
- **📦 발주 요약 자동 생성:** 오주문 방지를 위한 실시간 장바구니 요약
- **🚀 서버 전송 시뮬레이션:** 중복 발주 방지(Button Lock) 및 전송 후 세션 초기화 로직 탑재

## 🛠 실행 방법 (Local Run)
```bash
pip install -r requirements.txt
streamlit run app.py
